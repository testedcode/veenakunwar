# Admin Guide

## Login
The admin panel is accessible at `/admin`.
Authentication is handled via Firebase Auth. 
*(Note: Do not store your admin email/password in any git-tracked documents. Use a password manager).*

## Features
- **Sessions**: Add/edit/delete weekly Zoom sessions.
- **Products**: Add new products, configure 10-image carousels, setup size variants (200g, 500g, etc), and set health badges.
- **Coupons**: Activate % off bulk discounts.

All data is synchronized automatically with the Firebase backend.
