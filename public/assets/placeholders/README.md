# Placeholder Images

Add your banner and gallery images here:

- `banner1.jpg` - First hero slider banner
- `banner2.jpg` - Second hero slider banner  
- `banner3.jpg` - Third hero slider banner
- `profile1.jpg` - Profile/gallery image 1
- `profile2.jpg` - Profile/gallery image 2
- `gallery1.jpg` - Gallery image 1
- `gallery2.jpg` - Gallery image 2

## Image Requirements

- **Banners**: 1920x800px (or similar wide format)
- **Gallery**: 800x800px (square) or 1200x800px
- **Formats**: JPG, PNG, WebP
- **Size**: Keep under 500KB for faster loading

## How to Add Images

1. Add your images to this folder
2. Update `src/components/Slider/sliderData.js` for banners
3. Update `src/utils/images.js` for gallery images
4. Images will be automatically served from `/assets/placeholders/`
